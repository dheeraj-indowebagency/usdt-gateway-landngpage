# CoinPayUSDT — Landing Page

Professional B2B landing website for the CoinPayUSDT payment gateway.

## Tech Stack

- **Next.js 16** (App Router, TypeScript)
- **Tailwind CSS 4** for styling
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Sections

- Hero with animated USDT coin and payment flow visual
- Stats counter with animated numbers
- Features (Pay-In API, Pay-Out API, Security, Settlement, Reconciliation, Integration)
- How It Works (4-step integration flow with code examples)
- Networks (TRC20 & BEP20 comparison with animated flow diagrams)
- Industries (8 target verticals)
- Testimonials
- CTA with Telegram contact
- Responsive Navbar with mobile menu
- Footer with links and network badges
